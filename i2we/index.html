I2we

Aug 2020 - Nov 2020

I2WE is an iOS dating app I worked on with a team of three others for our Database Systems final project. We used Swift for the front end and Firebase as the backend to implement user registration and login, a profile page, a search page where users can search for other users based on certain criteria (i.e. age, school), and a fully-featured chat system (text, images, location, audio, video).
